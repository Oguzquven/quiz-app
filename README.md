🧠 JavaScript Quiz App
A vanilla JavaScript quiz application that tests your JavaScript knowledge with a timer, scoring system, and high score tracking.
<img width="1915" height="915" alt="image" src="https://github.com/user-attachments/assets/7f741eff-1167-46f0-a31a-76730e3f21f1" />

✨ Features

10 JavaScript knowledge questions
15-second countdown timer per question
Instant feedback — correct answers highlight green, wrong ones red
Score & result screen at the end
🏆 High score saved with localStorage
Fully responsive design
